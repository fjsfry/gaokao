## 一、项目概述

寻鹿升学是面向河北新高考家庭的志愿填报风险评估系统。网站围绕“提交前复核”这一核心场景，帮助学生和家长在正式提交志愿表之前，快速识别96个志愿中的冲稳保结构、梯度异常、保底不足、选科限制、专业限制、学费和偏好冲突等风险。

系统不承诺录取结果，也不把志愿填报包装成玄学。它的定位是把公开投档数据、位次区间、历史波动、规则限制、家庭偏好和AI报告生成整合到一个可复核的流程里，让用户知道哪些志愿可以保留、哪些需要下调、哪些需要替换、哪些必须人工进一步核验。

## 二、网站介绍

当前网站已经从早期单页展示升级为更正式的商业化产品网站，按照导航栏拆分为首页、产品介绍、在线评估、报告样例和服务方案几个主要页面。

首页承担品牌展示和核心转化功能，突出“河北新高考志愿风险评估系统”的定位，并提供免费生成预览报告的入口。产品介绍页集中说明系统如何读取志愿表、匹配公开数据、判断风险并生成报告。在线评估页用于输入考生分数、位次、选科、批次、偏好和志愿表，生成整表风险概览与逐条建议。报告样例页展示家长可理解的报告结构。服务方案页说明免费预览、单次报告、三次复查和填报季卡等服务模式。

## 三、服务对象

寻鹿升学当前主要服务以下几类用户：

1. 河北新高考考生家庭  
   适合已经形成志愿草稿、希望在提交前复核风险的学生和家长。系统可以帮助他们发现冲刺过多、稳妥承接不足、保底失效、专业限制、学费压力和地域偏好冲突等问题。

2. 高三家长  
   家长往往关心“孩子能不能录、有没有退档风险、要不要调整顺序”。网站通过可读报告和逐条建议，把复杂的位次与投档数据转成更容易沟通的风险提示。

3. 志愿填报咨询师  
   咨询师可以把系统作为前置体检工具，先让系统筛出高风险志愿和证据不足的条目，再集中处理最关键的问题，提高人工咨询效率。

4. 小型升学规划工作室  
   适合以微信、电话或线下咨询成交的服务团队。当前采用授权码模式，不要求客户注册账号，更适合顾问付款确认后直接发码使用。

5. 需要多轮复查的用户  
   对于反复调整志愿表的家庭，三次复查码和填报季卡可以支持多轮修改后的再次评估。

- 顾问沟通摘要生成。

### 7. DeepSeek完整报告集成

网站已经集成DeepSeek，用于在规则引擎生成风险结果后，进一步整理成面向家长可读的完整报告。

安全处理方式如下：

- DeepSeek API key只放在服务器环境变量中。
- 前端不直接接触DeepSeek密钥。
- 前端只调用服务端接口。
- 服务端根据规则引擎结果向DeepSeek请求完整报告。
- 生成完整报告前需要授权码验证。

### 8. 授权码商业化体系

网站已经取消传统登录系统，改为授权码模式。用户可以免费生成预览报告；付款后由顾问发送授权码，用户输入授权码后才可以生成完整报告。

当前支持三类授权码：

- 单次报告码：可生成1次完整报告。
- 三次复查码：可生成3次完整报告，适合多轮修改。
- 填报季卡：默认不限制总次数，但有每日生成上限，适合填报季反复复查。

该模式的优势是操作简单、适合微信成交、不需要客户注册账号，也方便通过手机向客户发码。
